🎯 WHAT YOU CAN DO

Dashboard (main hub at /dashboard):
  • View portfolio stats
  • See agent impact router panel
  • Switch between demo users
  • Browse trades & donations
  • Navigate to any feature

Change Where Profits Go:
  • Agent Panel → [Change Charity Destination]
  • Pick a different charity
  • Agent learns your preference
  • All future profits route there

Browse Charities:
  • Click [Marketplace] from dashboard
  • See 3 verified charities
  • Click [Donate Now]
  • Make testnet donation

View Impact:
  • Click [Impact] from dashboard
  • See all donations made
  • View impact metrics
  • Check leaderboards

🎮 INTERACTIVE FEATURES

Try These:

1. Switch Users
   Dashboard → Account dropdown → Select different user
   (Portfolio updates with their stats)

2. Change Charity
   Dashboard → Agent Panel → [Change Charity Destination]
   (Pick different charity, agent updates)

3. Browse Marketplace
   Dashboard → [Marketplace]
   (See 3 charities with detailed info)

4. Make Donation
   Dashboard → [Donate] or Marketplace → [Donate Now]
   (Would connect wallet and send testnet SOL)

5. View Impact
   Dashboard → [Impact]
   (See all donations and impact metrics)
