## 🤖 Agent System

### How It Works
```
User selects charity
    ↓
Agent initializes
    ↓
Agent monitors trades
    ↓
Profitable trade detected
    ↓
Agent calculates 5% of profit
    ↓
Funds routed to charity wallet
    ↓
Donation logged on dashboard
    ↓
Monthly impact updated
```

### Key Features
- ✅ Auto-detects profitable trades
- ✅ Calculates donation percentage
- ✅ Routes to selected charity
- ✅ Logs all transactions
- ✅ Updates metrics in real-time
- ✅ Never touches principal
- ✅ Never accesses private keys
