## 🎯 Key Takeaways

**What is this?**
- A Solana app that routes trading profits to charities
- Uses agentic automation
- Works on devnet (testnet)
- Fully functional demo

**What can you do?**
- Browse charities on marketplace
- Make testnet donations
- Use agent to auto-donate
- Switch between demo users
- View impact dashboard

**How do you start?**
1. `npm run dev`
2. Open http://localhost:3000
3. Click [Dashboard]
4. Explore

**What's it built with?**
- Next.js 14 (React)
- Tailwind CSS
- Solana Web3.js
- TypeScript
- Zustand

**When can you deploy?**
- Now (local)
- Today (docker)
- Today (vercel/railway)
- After setup (mainnet)
